# QC

A Vite + React + TypeScript starter for the QC quantum circuit simulator.

## About

This repo began as a lightweight `quantum_casino.html` Qiskit-style circuit simulator and is now migrated to a Vite React app.

## Running locally

```bash
pnpm install
pnpm dev
```

## Project structure

- `src/` — React application source
- `public/` — static assets
- `quantum_casino.html` — original simulator file preserved in the root
