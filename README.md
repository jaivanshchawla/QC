# QC

This project simulates quantum circuits inspired by Qiskit, implemented in plain HTML/JavaScript without requiring the Qiskit library.

## Files
- `quantum_casino.html` — main simulation file

## Setup
Open `quantum_casino.html` in a browser to run the simulation.
